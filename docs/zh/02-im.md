# 02 IM 消息、群组与媒体

[English](../en/02-im.md) | [返回中文索引](../README.md)

## 覆盖模块

- `feishu_bot_sdk.chat` -> `ChatService` / `AsyncChatService`
- `feishu_bot_sdk.im.messages` -> `MessageService` / `AsyncMessageService`
- `feishu_bot_sdk.im.media` -> `MediaService` / `AsyncMediaService`
- `feishu_bot_sdk.im.content` -> `MessageContent`（消息内容构造器）

## 快速示例

```python
from feishu_bot_sdk import ChatService, FeishuClient, FeishuConfig, MediaService, MessageService

client = FeishuClient(FeishuConfig(app_id="cli_xxx", app_secret="xxx"))
chat = ChatService(client)
message = MessageService(client)
media = MediaService(client)

sent = message.send_text(receive_id_type="open_id", receive_id="ou_xxx", text="你好")
print(sent.message_id)
image = media.upload_image("demo.png", image_type="message")
message.send_image(receive_id_type="open_id", receive_id="ou_xxx", image_key=image.image_key)

message.send_markdown(
    receive_id_type="open_id",
    receive_id="ou_xxx",
    markdown="### 日报\n\n任务完成",
)

groups = chat.list_chats(user_id_type="open_id", page_size=10)
print(groups.items)

announcement = chat.get_announcement("oc_xxx", user_id_type="open_id")
print(announcement.get("announcement_type"))

blocks = chat.list_announcement_blocks("oc_xxx", revision_id=-1, page_size=20)
print(blocks.items)
```

## `ChatService` API 一览

- 群组元数据：`create_chat`、`get_chat`、`list_chats`、`iter_chats`、`search_chats`、`iter_search_chats`、`get_share_link`、`update_chat`、`delete_chat`
- 群权限与置顶：`get_moderation`、`iter_moderation`、`update_moderation`、`put_top_notice`、`delete_top_notice`
- 成员与管理员：`add_members`、`remove_members`、`join_chat`、`list_members`、`iter_members`、`check_in_chat`、`add_managers`、`remove_managers`
- 群公告（新版块模型）：`get_announcement`、`list_announcement_blocks`、`iter_announcement_blocks`、`get_announcement_block`、`list_announcement_children`、`iter_announcement_children`、`create_announcement_children`、`batch_update_announcement_blocks`、`delete_announcement_children`
- 标签页与菜单：`create_tabs`、`update_tabs`、`sort_tabs`、`list_tabs`、`delete_tabs`、`create_menu`、`get_menu`、`update_menu_item`、`sort_menu`、`delete_menu`

说明：

- `get_announcement` 返回的是新版群公告元数据；后续内容读取和编辑都通过 block API 完成。
- `block_id` 通常可先通过 `list_announcement_blocks` 或 `list_announcement_children` 获取。
- 群公告写操作建议总是显式传 `revision_id`，需要幂等时再补 `client_token`。

## `MessageService` API 一览

- 发送与回复：`send_text`、`send`、`send_markdown`、`reply_text`、`reply_markdown`、`reply`
- 编辑与撤回：`edit`、`recall`、`get`
- 历史与已读：`list_history`、`query_read_users`
- 转发：`forward`、`forward_thread`、`merge_forward`
- 跟随气泡与 URL 预览：`push_follow_up`、`batch_update_url_previews`
- 表情与 Pin：`add_reaction`、`list_reactions`、`delete_reaction`、`pin_message`、`unpin_message`、`list_pins`
- 批量消息：`send_batch`、`send_batch_message`、`delete_batch_message`、`get_batch_message_progress`、`get_batch_message_read_users`
- 加急消息：`send_urgent_app`、`send_urgent_sms`、`send_urgent_phone`
- 卡片：`patch_card`、`send_ephemeral_card`、`delete_ephemeral_card`、`delay_update_card`
- 多类型快捷发送：`send_markdown`、`send_post`、`send_image`、`send_interactive`、`send_share_chat`、`send_share_user`、`send_audio`、`send_media`、`send_file`、`send_sticker`、`send_system`、`send_system_divider`

## `MessageContent` 构造器

- 基础：`text`、`image`、`file`、`media`、`audio`、`sticker`、`share_chat`、`share_user`
- 卡片：`interactive`、`interactive_card`、`interactive_template`
- 富文本 post：
  - 整体：`post`、`post_locale`
  - 元素：`post_text`、`post_link`、`post_at`、`post_image`、`post_media`、`post_emotion`、`post_code_block`、`post_hr`、`post_md`
- 系统消息：`system`、`system_divider`

## `MediaService` API 一览

- 图片：`upload_image`、`upload_image_bytes`、`download_image`
- 文件：`upload_file`、`upload_file_bytes`、`download_file`
- 消息资源下载：`download_message_resource`

## 常用片段

编辑消息（`msg_type` 默认 `text`）：

```python
message.edit("om_xxx", content={"text": "更新后的内容"})
```

批量发送：

```python
message.send_batch_message(
    msg_type="text",
    content={"text": "批量通知"},
    open_ids=["ou_a", "ou_b"],
)
```

下载消息中文件资源：

```python
data = media.download_message_resource(
    message_id="om_xxx",
    file_key="file_xxx",
    resource_type="file",
)
```

## 异步版

- `AsyncMessageService`、`AsyncMediaService` 与同步方法名保持一致。
- 仅调用方式改为 `await`。
